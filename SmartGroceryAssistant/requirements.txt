# Smart Grocery Shopping Assistant - Requirements

## Python Dependencies
- Python 3.7 or higher (built-in libraries only)

### Standard Library Modules Used:
- `json` - For data persistence
- `os` - For file operations
- `sys` - For system operations
- `datetime` - For date/time handling
- `typing` - For type hints
- `dataclasses` - For data models
- `collections` - For Counter and defaultdict
- `re` - For pattern matching

## Installation Instructions

### Option 1: Direct Run (Recommended)
1. Ensure you have Python 3.7+ installed
2. Download/clone the project to your local machine
3. Navigate to the project directory
4. Run: `python main.py`

### Option 2: Virtual Environment (Optional)
```bash
# Create virtual environment
python -m venv grocery_env

# Activate virtual environment
# On Windows:
grocery_env\Scripts\activate
# On macOS/Linux:
source grocery_env/bin/activate

# Run the application
python main.py
```

## No External Dependencies Required
This project is designed to run with Python's standard library only, making it easy to set up and run on any system with Python installed.

## System Requirements
- Python 3.7 or higher
- At least 10MB of free disk space
- Command-line interface support
- Operating System: Windows, macOS, or Linux

## File Permissions
The application needs read/write permissions to create and manage data files in the `data/` directory.
